### 运行

- git clone 
- 在HBuilderX编辑器中打开此项目
- 对项目点右键，可以识别项目类型 [详情](https://ask.dcloud.net.cn/article/35878)
- 打开终端：yarn
- 运行-运行到浏览器-chrome
- http://localhost:8080/ 

### 配置

- [api申请](https://oauth.cnblogs.com/)
- /common/request.js 补充client_id、client_secret

### 运行到微信小程序

- 运行-运行到小程序模拟器-微信开发者工具
